# AI模拟测试

## 场景：用户问"太阳星座为什么重要"

## 给AI的上下文：
```
[1] 【当代占星研究】太阳星座代表核心自我、人生目标
[2] 【星座第一书】太阳是英雄之旅的象征
```

## System Prompt要求：
- 使用 `<kb doc="DOCUMENT_NAME" />` 格式引用
- 每个事实必须引用

## AI的可能输出：

### 情况1：AI遵守指令（期望）
太阳星座代表一个人的核心自我和人生目标。<kb doc="当代占星研究" />
它是占星学中的"英雄之旅"。<kb doc="星座第一书" />

### 情况2：AI不遵守指令（当前问题）
太阳星座代表一个人的核心自我和人生目标。它是占星学中的"英雄之旅"。

## 问题分析：
如果AI输出情况2，说明：
1. AI忽略了system prompt中的引用要求
2. 或者AI不理解`<kb doc="..." />`格式
3. 或者AI认为这些是自己的知识，不是来自资料

## 根本原因：
AI可能没有正确识别出哪些信息来自检索资料，哪些是自己的知识。

## 解决方案：
在context中更明确地标注来源，比如：
```
[来源：当代占星研究] 太阳星座代表核心自我...
[来源：星座第一书] 太阳是英雄之旅的象征...
```

然后要求AI在回答中保留这些来源标记。
